# Modul 11 - Greedy Algorithm

## Subjek
Algoritma Greedy

## Deskripsi
Implementasi dari algoritma greedy untuk beberapa kasus:

### 1. Coin Change (Pembagian Koin)
- **File**: `CoinChange.py`
- **Deskripsi**: Algoritma greedy untuk mencari solusi pembagian koin minimum untuk jumlah tertentu

### 2. Fractional Knapsack (Tikar Pecahan)
- **File**: `FractionalKnapsack.py`
- **Deskripsi**: Algoritma greedy untuk mencari nilai maksimum yang dapat dibawa dengan kapasitas tas terbatas

### 3. Minimum Spanning Tree (Prim)
- **File**: `PrimMST.py`
- **Deskripsi**: Algoritma Prim untuk mencari Minimum Spanning Tree dari sebuah graf

### 4. Minimum Spanning Tree (Kruskal)
- **File**: `KruskalMST.py`
- **Deskripsi**: Algoritma Kruskal untuk mencari Minimum Spanning Tree dari sebuah graf
